## 📖 About

Preparing for the **IBPS SO IT Officer Scale I** exam meant hunting across a dozen coaching-site blogs, outdated YouTube playlists, and scattered PDFs just to answer basic questions: *What's the syllabus? What's the salary? How many vacancies? What do I study in Month 3?*

So I built the resource I wished existed — a single, calm, well-researched page that covers **everything an IT Officer aspirant needs** in one place.

No ads. No popups. No "Download now!" banners. Just the signal, with a Government of India aesthetic that feels appropriate to the subject matter.

---

## ✨ Features

- 📊 **Exam Overview** — conducting body, posts, selection stages, cycle dates (CRP SPL-XVI · 2026–27)
- 📝 **Full Exam Pattern** — Prelims (150 marks · 60 min) & Mains (60 questions · 45 min) with negative marking
- 📚 **Complete Syllabus** — interactive accordion covering all 10 Mains IT topics + Prelims reasoning/quant/English
- 📈 **IT Officer Vacancy History** — year-wise data from 2019–20 to 2026–27 (IT Officer seats only, not total SO)
- 🏦 **11 Participating Banks** — full list with correction callout on SBI's separate SCO recruitment
- 🗓️ **12-Month Roadmap** — month-by-month study plan with depth guidance (no LeetCode grinding required)
- 💰 **Salary & Career** — 12th Bipartite Settlement figures, pay scale, allowance breakdown, 16-year progression
- 🪜 **Promotion Ladder** — Scale I → Scale VII with realistic timelines
- 🧠 **Interactive Quiz** — 10 curated questions spanning DBMS, networks, OS, security, reasoning
- 📺 **18 Free Resources** — YouTube channels for every subject
- 📄 **9 PYQ Sources** — where to get previous year papers without paying
- ✍️ **Built with care** — by **Ishan Chakraborty**, an aspirant walking the same path

## 🎨 Design Philosophy

The UI follows an **authentic Government of India aesthetic** — not as a gimmick, but because the content deserves gravity:

| Element | Treatment |
|---------|-----------|
| **Colors** | Saffron `#FF9933` · Ashoka Navy `#000080` · India Green `#138808` |
| **Background** | Parchment cream `#FAF7F0` — paper-document feel |
| **Typography** | Merriweather (serif headings) · IBM Plex Sans (body) · Tiro Devanagari Hindi |
| **Symbols** | Ashoka Chakra emblem in the navbar + watermark in the hero |
| **Structure** | Tricolor strip pinned at the top, mirrored in the footer |

---

## 🛠️ Tech Stack

**Zero frameworks. Zero build step. One file.**

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations, no Tailwind
- **Vanilla JavaScript** — quiz logic, typewriter, accordion, scrollspy, animated bars
- **Google Fonts** — Merriweather, IBM Plex Sans/Mono, Tiro Devanagari Hindi
- **Vercel** — static hosting with GitHub auto-deploy

Everything lives in `index.html` — CSS in `<style>`, JS in `<script>`. You can read the whole site from top to bottom.

## 🚀 Running Locally

Want to fork this or customize it for a different exam?

```bash
# Clone the repo
git clone https://github.com/kujohishan/ibps-so-it-prep.git
cd ibps-so-it-prep

# Open in your browser — no build needed
open index.html
# or on Linux:
xdg-open index.html
# or just double-click the file
```

That's it. No `npm install`. No webpack. No tooling tax.
